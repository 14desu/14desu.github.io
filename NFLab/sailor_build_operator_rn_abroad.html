영국유학함장

 수리속도 이득이 있는 함장 육성법
 함장국적변경권(캐쉬템) 사용

　  국적변경 but 기존국적 어빌유지
　　  국적변경 Lv은 최종어빌에 영향이 없다
　　  Lv90 국적변경 ↔ Lv120 국적변경 : 최종어빌 동일
　  영국국적 = 수리보수특화 
　  캐쉬템가격 : KRW 30,000

 효과 : 수리어빌 +219

　  수리속도/s : 전함함장(빨랫줄조정) +1.6~1.9 / 항모함장 +3.6
　  Lv12~120 x 수리성장치+2 = 216 / Lv12 국적변경시 보너스 = 3
　  수리강화 +2 (수리어빌+240)에 근접하는 효율
　  수리강화 +1 추가적용가능

 캐쉬템 가격과 성능메리트를 비교해보고 판단하자

　  함선전체 수리속도/s : Max 280 상한선이 존재
　  세팅에따라 수리속도를 함장이외 수병으로도 상한선까지 확보가능
　  최종육성시의 함선전체의 수리속도를 계산기 활용해서 확인하자
　  이벤트보상에 함장국적변경권이 포함되기도 한다

  수병계산기 : 영국유학함장 계산가능
  함선 수리기관계산기 : 최종육성시 함선전체 수리속도 계산가능
