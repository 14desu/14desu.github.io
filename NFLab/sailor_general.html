수병능력

 수병능력 = Real어빌
 Real어빌 = 어빌 x ( 사관 x 4 + 숙련병 x 1 ) x 수병빈혈도

수병능력 ∝ 어빌
수병능력 ∝ ( 사관 x 4 + 숙련병 x 1 )
수병능력 ∝ 수병빈혈도
수병빈혈도 = ( 사관 + 숙련병 + 신병 ) / 수병수

 수병능력 ≠ 어빌 ( 관련이 있으나 일치하지 않는다 )
 갑판병 > 보정능력 ∝ only어빌 ↔ 수리/보수자체능력 ∝ Real어빌

이벤트수병 히든어빌


이벤트수병 (개근/전설보조/전설특무) 에는 히든어빌이 있다.
이로 인해 90렙 지급수병의 성장어빌↔누적어빌이 일치하지 않는다.
부어빌의 성장어빌은 +9고정이나 누적어빌은 644~1100의 편차를 가진다.
이는 1-90렙 구간의 성장치어빌이 +7~+12 랜덤로 반영되었기 때문이다.
이를 이벤트수병의 히든어빌이라 한다.

수병계산기에서 이벤트수병을 선택시 히든어빌을 반영한 계산을 지원한다.

사관사망과 강제종료


인게임에서 종종 사관/숙련병/신병이 죽는경우가 발생한다
- 발생조건 > 함포/어뢰의 관통판정시 적은 확률
- 수병사망시 > 특정사운드 + 우측하단 함장 인게임 메세지

전투룸을 나가기전에 강제종료를 하면 사관/숙련병이 사망하지 않는다
- 다만 이경우 해당 전투의 경험치 소득도 받지 못한다

 사관사망이 의심되면 강제종료를 하자
- 함장 메세지가 중첩될 경우 직전 메세지가 가리는경우가 있다
- 경험치는 게임한판 더 하면 되지만 사관은 돌아오지 않는다

　　  횟수제 프리미엄쉽 함장석에 갑판병 탑승 : 보정효과 발동

　  갑판어빌, 수병병종(갑판장/일등갑판병/이등갑판병), 수병국적이 관련
　  사관/숙련병, 수병빈혈도의 영향을 받지 않는다

　　  갑판잠재어빌 127 = 잠재능력 6% 보정, 갑판연사어빌 142 = 연사능력 6% 보정
　  복수탑승 페널티 = 1 /   (x = 갑판병탑승인원)
　  최대3명까지만 복수탑승적용, 4명이상 탑승시에는 3명만 적용
　  경우에 따라서는 복수탑승이 1명탑승보다 보정효율이 낮을 가능성이 존재

수병무게

 네이비필드 수병은 무게를 가진다
　  수병무게 : 함선배수량에 반영됨 = 함선속도, 장갑량에 영향을 미침
　　  렙업후 신병충원시 만재배수량초과로 출항불가한 경우가 존재
　　　　이럴때에는 방뢰 혹은 장갑을 1장 떼어주면 해결된다
　  수병수 x 국적계수 = 수병무게
　　  국적계수 : 미국0.53, 영국0.55, 독일0.565, 나머지0.5
　  의도적으로 수병수를 줄이는 경우가 있다
　　　영국폼폼이 : 연사플미 + 최대연사확보/다이어트
　　　영국갑판 : 수병0명
　　　영국함장 : 잠재플미 + 빨랫줄확보/다이어트
　　  목적 : 최대한 배수량 확보함으로 장갑or속도 이득
