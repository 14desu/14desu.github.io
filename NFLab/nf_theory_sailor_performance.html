네이비필드 연구실 / 수병능력


수병능력

 수병능력 = True어빌
 True어빌 = 누적어빌 x ( 사관 x 4 + 숙련병 x 1 ) x 수병빈혈도

수병능력 ∝ 누적어빌
수병능력 ∝ ( 사관 x 4 + 숙련병 x 1 )
수병능력 ∝ 수병빈혈도
수병빈혈도 = ( 사관 + 숙련병 + 신병 ) / 수병수

 수병능력 ≠ 성장어빌 ( 누적어빌에 관련이 있으나 일치하지 않는다 )
 수병능력 ≠ 누적어빌 ( True어빌에 관련이 있으나 일치하지 않는다 )
 갑판병 > 보정능력 ∝ only누적어빌 ↔ 수리/보수자체능력 ∝ True어빌
