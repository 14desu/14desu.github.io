수병능력

 수병능력 ≠ 성장어빌 ( 누적어빌에 관련이 있으나 일치하지 않는다 )
 수병능력 ≠ 누적어빌 ( True어빌에 관련이 있으나 일치하지 않는다 )
 갑판병 보정능력 ∝ 갑판병 어빌 (=사관&숙련병 영향x) ↔ 갑판병 자체 수리/보수능력 ∝ True ability (=사관&숙련병 영향o)


갑판병의 효과

 타수병의 능력보정

 동일함선탑승 타수병능력을 상향보정
　  타수병의 주어빌을 보정
　　  함장-잠재어빌, 기관병-기관어빌, 수리병-수리어빌
　  함장석 수병탑승 : 갑판병 보정효과의 발동조건
　　  횟수제 프리미엄쉽 함장석에 갑판병 탑승 : 보정효과 발동

 상향보정율 : 1% 단위 계단식구간
　  갑판어빌, 수병병종(갑판장/일등갑판병/이등갑판병), 수병국적이 관련
　  사관/숙련병, 수병빈혈도의 영향을 받지 않는다
　  어빌21당 1%보정 (갑판장일 경우, 전국가 동일)
　　  갑판잠재어빌 127 = 잠재능력 6% 보정, 갑판연사어빌 142 = 연사능력 6% 보정
　  복수탑승 페널티 = 1 /   (x = 갑판병탑승인원)
　  최대3명까지만 복수탑승적용, 4명이상 탑승시에는 3명만 적용
　  경우에 따라서는 복수탑승이 1명탑승보다 보정효율이 낮을 가능성이 존재
　  연합국/추축국 내에서 타국적보정가능 but 보정효율은 절반
　　  영국개근갑판으로 미국포병의 연사구간업 보정이 가능

 갑판병자체 수리능력/보수능력

타 병종과 동일하게 적용된다
  ∝어빌, ∝사관/숙련병, ∝수병빈혈도
갑판보정능력과 달리 함장석 수병미탑승시에도 적용된다
갑판병 고사관은 갑판병자체의 수리능력을 올리려는 목적
