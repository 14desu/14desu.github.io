측면장갑 0.2인치

 측면장갑 0.2인치 세팅의 예)


측면장갑 0.2인치 세팅의 이유 = 관통방어력1 확보
네이비필드 모든함선에 공통적으로 추천되는 기본세팅이다

 어뢰에 의한 수병사망방지

네이비필드의 어뢰는 관통력이 1미만이다
측면장갑의 관통방어력1을 확보하면 비관통판정이 되므로
어뢰에 의해서 수병사망이 발생하지 않는다

 어뢰데미지 감소

측면장갑0.2인치로 관통방어력1을 확보하면 비관통판정이 되므로
어뢰의 관통데미지를 막아냄으로서 어뢰데미지감소효과가 있다
측면장갑20인치이상세팅하더라도 데미지감소효과는 크게 증가하지 않는다
 측갑0.2인치 > 데미지감소 -40후반%, 측갑20인치이상 > 데미지감소 -50후반%
 적은투자-고효율인 0.2인치 관통방어력1 확보가 추천세팅

 급강하폭격기 매직도탄

급강하폭격기의 폭탄입사각이 측갑에서보면 90도에 가깝기 때문에
매직도탄을 기대해 볼수 있다 (낮은확률 / 가끔 운빨로 기대해줄 수준)

잠수함 산소충전 개념

① 수병 잠재능력

　  수병의 잠재어빌 x 수병구성(사관/숙련병) x 수병빈혈도 + 갑판보정
　  수병빈혈도 90% 미만이면 잠재능력0 판정
　  수병능력설명

② 수병 산소충전속도

　  잠재능력(=Real어빌)구간 → 산소충전속도 / 계단식구간
　　  잠재능력0 : 산소충전속도 0.48s /s (=1초당 0.48초 잠항가능산소충전)
　　  잠재능력5,750,000 : 산소충전속도 4.0s /s (=1초당 4.0초 잠항가능산소충전)
　  실데이터검증 → 계산공식검토완료 : 수병계산기 반영

③ 함체 잠항시간

　  함체 컴포넌트에 설정되어 있는 잠항시간(=산소탱크크기)
　  잠수함 잠항시간 정보

④ 산소충전시간

　  산소충전 최소단위 : 0.96s (= 24f) / 계단식구간
　  산소충전시간 = ( 산소탱크크기 / 산소충전속도 ) 최소단위올림
