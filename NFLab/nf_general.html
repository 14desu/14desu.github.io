연사속도단위

  네이비필드 : 25f 구동게임 > 1f = 0.04s(초)
 구현 연사속도단위 : 0.24s

네이비필드 상에서의 연사속도는 0.24s의 배수로 구현된다
> 최소 연사속도는 0.24s
연사속도가 최대로 다다른 경우 0.24 s로 일정하게 포격이 구현되고
이를 많은 유저들이 레이저대공이라고 부르고 있다

 평균 연사속도단위 : 0.04 s

최대연사구간에서 1구간 모자른 경우 위 사진과 같이 6연발이 된다
포탄간격을 잘 살펴보면 0.48s-0.24s-0.24s-0.24s-0.24s-0.24s가 되는데
이를 평균내보면 0.28s(7f)이고 최대연사 0.24(6f)보다 0.04s(1f)느리다
이와같이 수병에의한 연사속도는 평균연사속도로 나타나고 최소단위는 0.04s이다


연사속도개념

① 수병 연사능력

수병의 연사어빌 x 수병구성(사관/숙련병) x 수병빈혈도 + 갑판보정
 수병능력설명

② 수병 연사구간

수병구간이 1오를수록 1%씩 연사속도가 향상된다
계단식구간이므로 구간컷을 살짝 넘기는 세팅이 고효율
계산기를 통해서 구간컷을 넘기는데 필요한 갑판보정치를 확인가능

③ 함포 연사속도

함포 컴포넌트에 설정되어 있는 초기연사속도
Real어빌 0 수병으로 실험해보면 이 초기연사속도와 일치하지 않는다
(한국서버의 경우 x 130%, 글로벌서버의 경우 안해봐서 몰?루)
 공홈 쉽야드 함포정보

④ 평균 연사속도

함포 초기연사속도에 의존한 계단식 구간이 설정되어 있다
함포 초기연사속도가 높은 고인치 전함주포는 계단식구간이 없다
함포 초기연사속도가 낮은 저인치 대공포는 연사구간이 바뀌어도 동일연사인 계단식구간이 발생한다
예) 영국 5.25RP대공 - 개근명강포병 + 갑판없음 <-동일연사-> 연사18산타포병 + 연사18구갑2명

⑤ 구현 연사속도

평균 연사속도에 의존해서 구현 연사속도가 결정된다
이는 최소 구현연사속도단위가 0.24s 이기때문
예1) 평균 0.28s > 구현 0.24s 83%(5/6) + 0.48s 17%(1/6)
예2) 평균 2.36s > 구현 2.12s 17%(1/6) + 2.40s 83%(5/6)

①FCS 명중보너스

집탄에 매우 큰 영향을 미치는 요소

FCS 명중보너스가 높은 미국 > 우월한 집탄
FCS 명중보너스가 낮은 이탈리아 > 구제불가능한 집탄

②함포 명중보너스

집탄에 매우 큰 영향을 미치는 요소 but 히든스탯

저사관일 경우 영향 but 일정이상 어빌확보후에는 영향이 미미하거나 없다

- 영국 외 전국가 : 육성완료시 필요어빌충족, 신경쓸일 없음
- 영국 : 프리미엄명중수병 or NF개근수병으로 칼직 육성시에도 갑판병없이 필요어빌충족가능

　> 추천 : 명중초기14+1 or 명중초기14+1갑판

잠수함 산소충전 개념

① 수병 잠재능력

　  수병의 잠재어빌 x 수병구성(사관/숙련병) x 수병빈혈도 + 갑판보정
　  수병빈혈도 90% 미만이면 잠재능력0 판정
　  수병능력설명

② 수병 산소충전속도

　  잠재능력(=Real어빌)구간 → 산소충전속도 / 계단식구간
　　  잠재능력0 : 산소충전속도 0.48s /s (=1초당 0.48초 잠항가능산소충전)
　　  잠재능력5,750,000 : 산소충전속도 4.0s /s (=1초당 4.0초 잠항가능산소충전)
　  실데이터검증 → 계산공식검토완료 : 수병계산기 반영

③ 함체 잠항시간

　  함체 컴포넌트에 설정되어 있는 잠항시간(=산소탱크크기)
　  잠수함 잠항시간 정보

④ 산소충전시간

　  산소충전 최소단위 : 0.96s (= 24f) / 계단식구간
　  산소충전시간 = ( 산소탱크크기 / 산소충전속도 ) 최소단위올림


측면장갑 0.2인치

 측면장갑 0.2인치 세팅의 예)


측면장갑 0.2인치 세팅의 이유 = 관통방어력1 확보
네이비필드 모든함선에 공통적으로 추천되는 기본세팅이다

 어뢰에 의한 수병사망방지

네이비필드의 어뢰는 관통력이 1미만이다
측면장갑의 관통방어력1을 확보하면 비관통판정이 되므로
어뢰에 의해서 수병사망이 발생하지 않는다

 어뢰데미지 감소

측면장갑0.2인치로 관통방어력1을 확보하면 비관통판정이 되므로
어뢰의 관통데미지를 막아냄으로서 어뢰데미지감소효과가 있다
측면장갑20인치이상세팅하더라도 데미지감소효과는 크게 증가하지 않는다
 측갑0.2인치 > 데미지감소 -40후반%, 측갑20인치이상 > 데미지감소 -50후반%
 적은투자-고효율인 0.2인치 관통방어력1 확보가 추천세팅

 급강하폭격기 매직도탄

급강하폭격기의 폭탄입사각이 측갑에서보면 90도에 가깝기 때문에
매직도탄을 기대해 볼수 있다 (낮은확률 / 가끔 운빨로 기대해줄 수준)
