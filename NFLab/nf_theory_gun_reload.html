함포 연사속도 개념

① 수병 연사능력

수병 연사능력 = 연사True어빌 = 연사누적어빌 x ( 사관 x 4 + 숙련병 ) x 수병빈혈도 + 갑판보정
 수병능력설명

② 수병 연사구간

수병구간이 1오를수록 1%씩 연사속도가 향상된다
계단식구간이므로 구간컷을 살짝 넘기는 세팅이 고효율
계산기를 통해서 구간컷을 넘기는데 필요한 갑판보정치를 확인가능

③ 함포 연사속도

함포 컴포넌트에 설정되어 있는 초기연사속도
True어빌 0 (=사관0명,숙련병0명) 수병으로 실험해보면 이 초기연사속도와 일치하지 않는다
(한국서버의 경우 x 130%, 글로벌서버의 경우 안해봐서 몰?루)
 공홈 쉽야드 함포정보

④ 평균 연사속도

함포 초기연사속도에 의존한 계단식 구간이 설정되어 있다
함포 초기연사속도가 높은 고인치 전함주포는 계단식구간이 없다
함포 초기연사속도가 낮은 저인치 대공포는 연사구간이 바뀌어도 동일연사인 계단식구간이 발생한다
예) 영국 5.25RP대공 - 개근명강포병 + 갑판없음 <-동일연사-> 연사18산타포병 + 연사18구갑2명

⑤ 구현 연사속도

평균 연사속도에 의존해서 구현 연사속도가 결정된다
이는 최소 구현연사속도단위가 0.24s 이기때문
예1) 평균 0.28s > 구현 0.24s 83%(5/6) + 0.48s 17%(1/6)
예2) 평균 2.36s > 구현 2.12s 17%(1/6) + 2.40s 83%(5/6)


함포 연사속도 단위

  네이비필드 : 25f 구동게임 > 1f = 0.04s(초)
 구현 연사속도단위 : 0.24s

네이비필드 상에서의 연사속도는 0.24s의 배수로 구현된다
> 최소 연사속도는 0.24s
연사속도가 최대로 다다른 경우 0.24 s로 일정하게 포격이 구현되고
이를 많은 유저들이 레이저대공이라고 부르고 있다

 평균 연사속도단위 : 0.04 s

최대연사구간에서 1구간 모자른 경우 위 사진과 같이 6연발이 된다
포탄간격을 잘 살펴보면 0.48s-0.24s-0.24s-0.24s-0.24s-0.24s가 되는데
이를 평균내보면 0.28s(7f)이고 최대연사 0.24(6f)보다 0.04s(1f)느리다
이와같이 수병에의한 연사속도는 평균연사속도로 나타나고 최소단위는 0.04s이다
