 영국 포병 일반론

　  영국 포병은 명중거포병/연사거포병/일등대공포병의 3종류 트리가 있다
　　  명중거포병트리 : 사용하지 않는다 (명중어빌특화트리 but 연사속도손해가 극심)
　　  연사거포병트리 : 일정명중어빌을 확보가 필수 → 개근수병 고효율
　　　  주포병 : 명중초기14칼직 + (명중1강화 or 1갑판탑승) 으로 타국동급집탄
　　　　  이후로 명중어빌을 추가로 확보해도 집탄에 대한 영향이 미미하거나 없다
　　　　   영국 집탄모음
　　　　   집탄움짤분석/각국비교
　　　  부포병 : 일반수병 연사12/수리11 >> 프리미엄 연사수병
　　　　  대공포 활용 5.25rp N형 : 연사초기 15 / 0.28s = 연사초기 14 / 0.28s 로 실연사속도차이가 없다
　　　　  일반수병 연사12+2/수리11 이 프리미엄 연사수병 연사14/수리9+1 대비 성능상 상위호환
　　  일등대공포병트리 : 특정 대공포 활용시에 사용, 늦직육성필수, 개인취향
　  연사14+1 포병+ 3갑판으로 주포연사up but 부포연사는 오르지 않는다
　　  프리미엄연사수병 + 3갑판시 명중능력확보가 불가능하여 타국대비 매우 모자란 함포 집탄
　　  3갑판 주포연사속도up은 명중초기성장치가 높은 개근수병에 한정하여 유효한 수병구성
　  결과적으로 주포는 연사거포병only, 부포는 연사거포병 or 일등대공포병 개인취향선택
　　  연사거포병 = 되도록이면 개근수병으로 육성 (매우 고효율)
　　  연사거포병(개근수병) : 추천 0갑판8기관
　　  연사거포병(프리미엄명중수병) : 추천 1갑판7기관
　　  일등대공포병 : 필요한 연사어빌이 매우낮음, 연사거포병에 맞춰서 갑판병구성을 하자

 연사거포병 / 늦전직 고찰

　  1차 명연패치로 영국 집탄하향  2008/04/08 패치노트
　　  여담이지만 패치노트에서 예고한 2차 명연패치는 2023년1월 현재 이루어지지 않고 있다
　  당시엔 300사관이면 상당한 고사관 but 45%만사관 대비 적은 사관숫자로 집탄에 필요한 명중어빌부족
　　  아직 40%/45% 사관템이 없던시절이었다  2009/06/03 패치노트
　  늦전직을 통하여 부족한 명중어빌을 확보하고자 했다
　　  명중성장 손해가 큰 중포병전직 타이밍을 조절 → 명중어빌확보 + 연사손해/수병수손해 최소화
　　  늦전직에도 육성법이 있지만 프리미엄명중수병+명중1강화 늦전직이 어빌손해가 제일 적다

　  늦전직이 의미를 가지려면 칼전직수병의 명중능력이 부족해야 한다
　　  하지만 칼전직수병으로도 일정이상의 집탄군이 형성되고, 명중능력을 추가확보해도 효과가 미미하다

　  영국 연사거포병은 되도록이면 [개근수병]으로 육성추천 (매우 고효율)

 연사구간&연사속도 / 일등대공포병 / 산타갑판제외

　  일명 폼폼이 Lv45 2pdr 1.57"/39 Mark VIII 전용병종, 2가지 육성방향이 있다

　　　   이등대공포병 전직시점을 최대한 늦추는게 무조건 이득 → 이등대공포병 만렙직 필수
　　　   연사초기9 늦전직수병(병기병Lv25,그외Lv120) 연사능력 >>> 연사초기14+1 칼직수병 연사능력
　　  수리능력 최대화(수리속도이득) : 수리플미 + 병기병Lv25/이등대공포병Lv120 = 수병수745
　　　  연사능력 최대화(빈혈수병 배수량이득) : 연사플미 + 병기병Lv12/이등대공포병Lv120 = 수병수758

 추천육성&대체육성 / 45%만사관&풀숙련병

　  영국 = 개근수병으로 육성하는 포병이 매우매우 고효율인 국가
　　  개근수병 = 고연사 0갑판8기관 가능 > 1갑판 강제되는 타국대비 긴 오버힛시간 > 영폭도
　  추천육성 : 개근포병 / 0갑판8기관
　  대체육성1 : 개근포병을 프리미엄포병+프리미엄갑판으로 대체 / 1갑판7기관
　　  1갑판 최대효율은 부포병 일반수병 연사12 수리11+2
　　  추후 개근포병 육성계획이 있으면 처음부터 일반수병 연사12+2 수리11로 육성
　  대체육성2 : 부포병 수리폼폼이로 대체
　　  폼폼이 함포가 가볍기 때문에 배수량이득 > 떡장함의 세팅의 자유도 up
　  대체육성3 : 개근포병+3갑판병 연사구간up 힙스터세팅 / 3갑판5기관
　　  3갑판사용시 오버힛시간손해를 감수해야 한다
