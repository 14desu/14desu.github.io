 미국 포병 일반론

　  미국 포병은 거포병트리와 대공병장트리가 있다
　　  미국 거포병트리는 경포/중포/거포에서 각각 명중전직or연사전직 선택 (교차전직가능)
　　　  미국 사통(FCS) 보정치가 타국대비 우월
　　　　  명11 연연연전직 포병으로도 매우 좋은 집탄 → 개근수병 비효율
　　  미국 대공병장트리는 사용하지 않는다 (대공어빌특화트리 but 대공어빌이 무쓸모)
　  결과적으로 미국 포병은 거포병만 사용, 경포/중포/거포전직은 연연연으로 고정
　　  연사거포병으로 전함함포 / 대공포 양쪽모두 운용한다

영국유학함장

 수리속도 이득이 있는 함장 육성법
 함장국적변경권(캐쉬템) 사용

　  국적변경 but 기존국적 어빌유지
　　  90렙 국적변경 ↔ 120렙 국적변경 : 최종어빌은 동일하다
　  영국국적 = 수리보수특화 
　  캐쉬템가격 : KRW 30,000

 효과 : 수리어빌 +219

　  수리속도/s : 전함함장(빨랫줄조정) +1.6~1.9 / 항모함장 +3.6
　  Lv12~120 x 수리성장치+2 = 216 / Lv12 국적변경시 보너스 = 3
　  수리강화 +2 (수리어빌+240)에 근접하는 효율
　  수리강화 +1 추가적용가능

 캐쉬템 가격과 성능메리트를 비교해보고 판단하자

　  함선전체 수리속도/s : Max 280 상한선이 존재
　  세팅에따라 수리속도를 함장이외 수병으로도 상한선까지 확보가능
　  최종육성시의 함선전체의 수리속도를 계산기 활용해서 확인하자
　  이벤트보상에 함장국적변경권이 포함되기도 한다
